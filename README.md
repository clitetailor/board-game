Board game
==========

Prerequisites
-------------

- MongoDB Driver

Setup
-----

```bash
npm i
```

Run
---

```bash
npm start
```

Folder Structure
----------------

```
- /app				<- client
	- /assets
	- /cmp					<- web components
	- /utils				<- utilities

	+ App.jsx			<- main page
	+ App.html
	+ App.styl

+ index.js			<- server

+ .gitignore
+ package.json			<- manifest file
+ webpack.config.js		<- build config file
```
